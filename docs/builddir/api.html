competitive_score(df: DataFrame, metrics_column: list, strategy='standard')[source]¶

Given the dataframe, and the columns with the metrics. Calculates the score by ranking the metrics. The first column of the dataframe must be the variables column. All others must be numeric.

df: pd.DataFrame

Dataframe containing the metrics.

metrics_column: list

List of columns containing the metrics.

strategy: str, optional (default: ‘standard’)

The scoring strategy to use. Can be ‘standard’ or ‘cluster’.
